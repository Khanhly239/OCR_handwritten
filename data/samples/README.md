# Sample images

Đặt các ảnh chữ viết tay tiếng Việt vào thư mục này để test pipeline.

Định dạng hỗ trợ: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.webp`.

`sample.png` là một ảnh "in" sinh tự động (không phải chữ viết tay) — chỉ dùng
để smoke test pipeline có chạy không. Để đo chất lượng thật, hãy thay bằng
ảnh chữ viết tay thực tế hoặc ảnh từ các dataset công khai:

- **VNOnDB** (Vietnamese Online Handwriting Database)
- **InkData** (HUST)
- **Cinnamon AI Marathon** (handwriting OCR challenge)
